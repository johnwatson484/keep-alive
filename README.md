# Keep Alive

Keep service endpoints active to prevent cool down

## Prerequisites

Either:
- Docker
- Docker Compose

Or:
- Node 20

## Running application
### Docker
```
docker-compose build
docker-compose up
```

### Node
```
node app
```
Or:
```
nodemon app
```
